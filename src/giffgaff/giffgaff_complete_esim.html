OAuth 2.0 登录
如何获取回调URL：
  1. 打开开发者工具：按 F12 或右键选择"检查"
  2. 切换到"网络"标签页
  3. 在登录页面输入邮箱验证码
  4. 点击登录按钮
  5. 在网络面板中查找 giffgaff://auth/callback/ 开头的请求
  6. 复制完整的URL 并粘贴到下方输入框

💡 提示：如果看不到网络请求，请先打开开发者工具再进行登录操作

回调URL格式示例：

giffgaff://auth/callback/?code=ABC123&state=XYZ789

确保URL包含 code 和 state 参数

Cookie 登录
Cookie登录说明：

Cookie登录功能需要PHP后端支持。系统会自动检测是否可用：

  • 🟢 支持PHP的服务器：功能完全可用
  • 🟡 静态托管（如Netlify）：自动回退到OAuth登录

推荐使用OAuth 2.0登录（适用于所有环境）：

  • ✅ 通用兼容 - 在任何部署环境都可用
  • ✅ 更安全 - 标准OAuth 2.0协议
  • ✅ 无需后端 - 纯前端实现

💡 点击"验证Cookie"按钮将自动检测当前环境是否支持

如何获取Cookie：
  1. 访问 giffgaff.com 并登录您的账户
  2. 打开开发者工具：按 F12 或右键选择"检查"
  3. 切换到"应用程序"标签页（Chrome）或"存储"标签页（Firefox）
  4. 在左侧选择"Cookie" → https://www.giffgaff.com
  5. 复制所有Cookie 或重要的认证Cookie

💡 Cookie格式：name1=value1; name2=value2; ...
