# 文档模块

## 模块职责

提供项目使用指南、架构说明、安全指南和问题修复记录。

## 文档结构

| 目录 | 内容 | 文件格式 |
|------|------|----------|
| `guides/` | 使用指南 | Markdown |
| `reference/` | 技术参考文档 | Markdown |
| `fixes/` | 问题修复记录 | Markdown |

## 核心文档

| 文档 | 路径 | 说明 |
|------|------|------|
| **架构说明** | `ARCHITECTURE.md` | 项目架构详解 |
| **安全指南** | `SECURITY.md` | 安全措施说明 |
| **Giffgaff 教程** | `User_Guide.md` | 图文操作指南 |
| **Giffgaff 英文教程** | `User_Guide_EN.md` | 英文版使用指南 |
| **Giffgaff 参考** | `reference/README_giffgaff_esim.md` | Giffgaff 技术文档 |
| **Simyo 参考** | `reference/README_simyo_esim.md` | Simyo eSIM 操作指南 |
| **CORS 解决方案** | `guides/CORS_SOLUTION.md` | 跨域问题解决方案 |
| **通知系统** | `guides/notification-system.md` | 通知功能说明 |

## 文档规范

- **语言**: 中文为主，英文版本并行维护
- **格式**: Markdown (CommonMark)
- **链接**: 使用相对路径，确保 GitHub 渲染正确
- **参考文档**: `reference/` 目录下的技术文档已精简，仅保留核心架构和 API 信息，详细使用流程请参考根 README
