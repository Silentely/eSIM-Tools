# 文档模块

## 模块职责

提供项目使用指南、API 参考、架构说明和问题修复记录。

## 文档结构

| 目录 | 内容 | 文件格式 |
|------|------|----------|
| `guides/` | 使用指南 | Markdown |
| `reference/` | API 参考和技术文档 | Markdown |
| `fixes/` | 问题修复记录 | Markdown |
| `images/` | 文档图片资源 | PNG/JPG |

## 核心文档

| 文档 | 路径 | 说明 |
|------|------|------|
| **Giffgaff 使用教程** | `guides/User_Guide.md` | 图文及视频操作指南 |
| **Giffgaff 英文教程** | `guides/User_Guide_EN.md` | 英文版使用指南 |
| **Simyo 使用教程** | `reference/README_simyo_esim.md` | Simyo eSIM 操作指南 |
| **Giffgaff 详细说明** | `reference/README_giffgaff_esim.md` | Giffgaff 技术文档 |
| **架构说明** | `ARCHITECTURE.md` | 项目架构详解 |
| **CORS 解决方案** | `guides/CORS_SOLUTION.md` | 跨域问题解决方案 |

## 文档规范

- **语言**: 中文为主，英文版本并行维护
- **格式**: Markdown (CommonMark)
- **图片**: 存放在 `images/` 目录，使用相对路径引用
- **链接**: 使用相对路径，确保 GitHub 渲染正确
